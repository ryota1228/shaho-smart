CSVインポートガイド

以下の手順に従って、従業員情報をCSV形式で一括登録できます。

📌 インポート手順

  1. 「一括登録用テンプレート」ボタンを押してCSVテンプレートをダウンロードします。
  2. テンプレートをGoogleスプレッドシートやExcelなどで開きます。
  3. 以下の項目に従って、各列を正しい形式で入力します。（1行=従業員1名）
    ※ヘッダー行（項目名）は削除しないでください
  4. 保存したCSVファイルを「CSVで従業員を一括登録」ボタンからアップロードします。

🟧 必須項目

項目 {{ element.field }} 形式 {{ element.type }} 形式例 {{ element.format }} 補足 {{ element.note }}
